Privacy Policy

Data Storage

No user data or uploaded files are permanently stored by this tool. Uploaded files are used solely for processing and analysis during the current session.

File Retention

All uploaded files and associated data are automatically deleted when the browser tab is closed, the browser is closed, or the session ends. No copies of uploaded files are retained after the session.

Data Sharing

Uploaded files and generated results are not shared with, transferred to, or accessed by any third party.

Disclaimer

Tool Status

This tool was developed as a personal initiative to support faculty members in assessing and analyzing Course Learning Outcome (CLO) attainment. It is not an official university system and has not been formally endorsed or approved by the university.

User Responsibility

While every effort has been made to ensure the accuracy of the calculations and analyses, users are responsible for reviewing and verifying all generated results before using them in official reports, accreditation documents, or decision-making processes.
